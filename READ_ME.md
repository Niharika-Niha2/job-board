# Job Board

A modern Job Board web application built using **Next.js**, **React**, and **Tailwind CSS**. The application allows users to browse jobs, filter listings, view job details, apply for jobs, and access a simple job posting interface.

## Live Demo

https://job-board-pi-roan.vercel.app/

## GitHub Repository

https://github.com/Niharika-Niha2/job-board

---

## Features

- Home page with modern UI
- Search jobs by title or company
- Filter jobs by location
- Filter jobs by job type
- Featured job listings
- Dynamic job details page
- Apply Now page
- Post Job page
- Responsive design
- Footer with navigation
- GitHub Actions CI pipeline
- Deployed on Vercel

## Installation

```bash
git clone https://github.com/Niharika-Niha2/job-board.git

cd job-board

npm install

npm run dev
```

Open:

```
http://localhost:3000
```
